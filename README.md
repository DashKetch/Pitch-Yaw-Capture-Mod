# 📸 Pycap: Pitch & Yaw Capture

![Minecraft Version](https://img.shields.io/badge/Minecraft-1.21.10-brightgreen)
![Fabric](https://img.shields.io/badge/Loader-Fabric-tan)
![License](https://img.shields.io/badge/License-MIT-blue)

Ever needed to lock in that **perfect** angle for a build, a cinematic shot, or a precise ender pearl throw? **Pycap** is a lightweight, high-performance Fabric mod designed for Minecraft 1.21.10 that lets you save and snap your camera orientation instantly! 🎯

---

### ✨ Features

* **💾 Instant Recording:** Save your exact Pitch and Yaw with a single keystroke.
* **🔄 Precision Snapping:** Snap your camera back to your saved orientation perfectly.
* **🛠️ Custom Category:** Your keybinds stay organized in their own `Pycap` category in the controls menu.
* **🔇 Stealth Mode:** Don't like chat spam? Toggle the action-bar notifications on or off!
* **⚡ Ultra Lightweight:** Built with the latest Fabric API for 1.21.10.

---

### ⌨️ Controls (Default)

| Action | Keybind |
| :--- | :--- |
| **Record Rotation** | `O` |
| **Snap to Rotation** | `P` |

> You can rebind these at any time in **Options > Controls > Key Binds** under the **Pycap** section!

---

### ⚙️ Config

Pycap comes with a simple built-in configuration system. 

* **Manual Config:** Located at `.minecraft/config/pycap.json`.

---

### 🚀 Installation

1.  Ensure you are on **Minecraft 1.21.10**.
2.  Install the latest **Fabric Loader**.
3.  Drop **Pycap** and the **Fabric API** into your `mods` folder.
4.  Start the game and take control of your camera! 🎥
